# react-dev-environment

# Set up

## Dependencies
- Babel
- Eslint
- React
- React-Redux
- React-Router
- React-Router-Redux
- Redux
- Redux-Thunk
- Webpack

## Installation
1. `npm install`
2. `npm start`

## Clone
`cp -r .gitignore .babelrc .eslintrc.yml package.json webpack.config.js README.md src assets ${your destination folder}`
